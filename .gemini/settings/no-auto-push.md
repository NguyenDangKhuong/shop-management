

# 🚫 No Auto-Push Rule

> **NEVER push to remote (git push) without explicit user approval.**

## Required

- Always ask the user before running `git push`
- Commit locally is OK, but pushing requires confirmation
- Wait for user to say "push" or "đẩy lên" before pushing

## Example

```
✅ "Đã commit xong. Bạn muốn push lên không?"
❌ Tự động chạy git push mà không hỏi
```
